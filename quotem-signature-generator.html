Logo is embedded so it always shows; for Outlook desktop at scale, host quotem-icon.png and swap the logo src for a URL.

How to install (Gmail & Outlook)
  1. Gmail: click Copy signature → Settings (gear) → See all settings → General → Signature → paste → Save changes.
  2. Outlook (web): Settings → Mail → Compose and reply → paste into the signature box.
  3. Outlook (desktop): use Copy signature and paste into File → Options → Mail → Signatures. If the logo drops, host the image and use the HTML version.
  4. Bulk deploy: use Copy HTML per employee, or hand IT the pattern to script from a staff list.
