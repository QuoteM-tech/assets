## Setup
1. Create a **public** repo named `assets` under the `QuoteM-tech` org.
2. Upload the `logo/`, `icons/`, and `qr/` folders (keep this structure).
3. Confirm a raw URL opens the image directly in a browser (not a GitHub page).
